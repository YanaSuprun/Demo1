Input Parameters: Number

Lucky tickets

There are 2 ways to count lucky tickets:

1. Simple - if the sum of the first three digits is equal to the sum of the last three, then this ticket is considered lucky.

2. Complicated - if the sum of the even numbers of the ticket is equal to the sum of the odd numbers of the ticket, then the ticket is considered lucky.

Determine which way will give a greater number of lucky tickets at a given interval

Input parameters: min and max(six digit numbers)

Output: an object with information about the winning method and the number of lucky tickets.
